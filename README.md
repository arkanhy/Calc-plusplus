# Calc++
A simple and lightweight CLI-based arithmetic calculator by me, Arkan H. Y.  

## Features
-Addition  
-Subtraction  
-Multiplication  
-Division  
-Percentage(%) to Decimal Converter  

## How to compile
cd src  
g++ Calc++_v1.1.cpp -o calc++  
sudo cp calc++ /usr/local/bin/  

## How to run
calc++
