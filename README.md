# Calc++
A simple and lightweight CLI-based arithmetic calculator by me, Arkan H. Y.  

## Features
-Addition  
-Subtraction  
-Multiplication  
-Division  
-Percentage(%) to Decimal Converter  
-Square Root  
-Exponent  
-Quadratic Formula  
-BMI Calculator  
-Length Converter  

## How to compile
cd calcplusplus  
sudo ./install.sh  
## How to run
calc++
